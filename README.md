<h1>Number Guessing Game</h1>
<h5><i>A Python version of Number Guessing game from the terminal!</i></h5>
<h5>There are 2 levels as "easy" and "hard". "Easy" level will have 10 guesses and "Hard" will be having 5 guesses. When you guess the number it'll prompt if the guessed number is too high or too low!</h5>
To Play: 
<h6></h6>

<ul>
  <li>Open your command prompt.</li>
  <li>Type <strong>python main.py</strong> or <strong>py -3 main.py</strong> and Enjoy the game!</li>
</ul>
